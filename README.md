## General information
This repository contain data collected from  135 studies that validate satellite-based ETa estimates with Eddy Covariance measurements, and a Jupyter notebook used for a meta-analysis of reported PBIAS in validation results.

## Data organisation and naming

The dataset is provided in /RSETValidationPapers_with_PBias_clean.xlsm

The Python scripts for meta-analysis are provided in /PBias_meta_analyses.ipynb

The outputs of the meta-analysis are provided in /Outputs
